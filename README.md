
# AI Shopper – AI Based E-Commerce Website

AI Shopper is a frontend based e-commerce website designed to provide a smart and modern online shopping experience.  
The website shows products, allows users to add items to cart or wishlist, and provides AI based product recommendations.

## Project Description

This project demonstrates how an e-commerce platform works on the frontend side.  
Users can browse products, filter them, and interact with the interface easily.

The goal of this project is to create a clean and responsive shopping interface.

## Features

- Modern UI design
- Product listing
- AI recommended products section
- Category filter
- Price filter
- Add to cart functionality
- Wishlist system
- Product modal view
- Responsive layout

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Structure
ai-ecommerce
│
├── assets/images → Images used in website
├── css/style.css → Website styling
├── js/main.js → Main website logic
├── js/products.js → Product data
├── js/cart.js → Cart functionality
├── js/wishlist.js → Wishlist functionality
├── index.html → Main homepage
├── cart.html → Cart page
└── wishlist.html → Wishlist page


## How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in a browser

The website will run locally without any server.
